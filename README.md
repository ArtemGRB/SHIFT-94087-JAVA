# Сортировка содержимого файлов по типам данных

##  Описание проекта

Консольное приложение для фильтрации содержимого текстовых файлов по типам данных. Программа считывает данные из одного или нескольких входных файлов, распределяет строки по типам (целые числа, вещественные числа, строки), записывает их в соответствующие выходные файлы и собирает статистику.

##  Возможности

-  **Обработка нескольких файлов** - можно указать любое количество входных файлов
-  **Три типа данных**:
    - Целые числа (INTEGER)
    - Вещественные числа (FLOAT)
    - Строки (STRING)
-  **Режимы статистики**:
    - Без статистики
    - Краткая статистика (количество элементов)
    - Полная статистика (минимум, максимум, сумма, среднее для чисел; длина для строк)
- ️ **Гибкая настройка**:
    - Настройка выходной директории
    - Префикс имен выходных файлов
    - Режим добавления к существующим файлам
- ️ **Обработка ошибок**:
    - Проверка корректности аргументов командной строки
    - Обработка ошибок чтения/записи файлов
    - Предупреждения о неизвестных опциях

## Требования к окружению

* **Java**: версия 17 или выше.
* **Система сборки**: Maven.



## Сборка проекта

```bash
mvn clean package
```

##  Использование


### Базовый синтаксис

```bash
java -jar util.jar [опции] входные_файлы...
```

### Примеры использования:

##### 1. Базовая обработка файлов:

```bash
java -jar target/SHIFT-94087-JAVA-1.0-SNAPSHOT.jar [опции] входные_файлы...
```

##### 2. С сохранением в указанную директорию:

```bash
java -jar target/SHIFT-94087-JAVA-1.0-SNAPSHOT.jar -o ./output in1.txt in2.txt
```

##### 3. С префиксом имен файлов и краткой статистикой:

```bash
java -jar target/SHIFT-94087-JAVA-1.0-SNAPSHOT.jar -p result_ -s in1.txt in2.txt
```

##### 4. С полной статистикой и режимом добавления:

```bash
java -jar target/SHIFT-94087-JAVA-1.0-SNAPSHOT.jar -a -f in1.txt in2.txt
```

## Параметры командной строки:

- **-o  PATH	Директория для сохранения результатов (по умолчанию: текущая директория)**
- **-p  PREFIX	Префикс имен выходных файлов**
- **-a	Режим добавления к существующим файлам (по умолчанию: перезапись)**
- **-s	Режим краткой статистики**
- **-f	Режим полной статистики**

### Примечания:

- Флаги -s и -f взаимоисключающие, приоритет у -f
- Если не указан ни -s, ни -f, статистика не выводится
- Аргументы без флагов интерпретируются как имена входных файлов
 

## Пример работы:
 ### Входные данные (input.txt):

```bash
  text
  123
  45.67
  hello world
  -890
  3.14e-10
  Тест
```  

 ### Команда:
  ```bash
  java -jar target/SHIFT-94087-JAVA-1.0-SNAPSHOT.jar -o output -p result_ -f input.txt
  ```

 ### Результаты:
 #### Файлы в директории output/:

- result_integers.txt: содержит 123 и -890

- result_strings.txt: содержит hello world и Тест
